## Simple-#Survey-Application.
This is a Simple Survey Application that allow users to respond to survey questions and view a list of all responses to the survey
questions.The application sends and fetches information fetches data to and from the database using a rest api known as django rest framework.The application utilzes features such search and pagination.
## Installation.
